Splats

Here you can send a request to the splatter service to generate a 3D Gaussian splatting model from an already-uploaded observation video. You can adjust various processing settings.

Step Setting Explanation
Observation ID
File ID
extract FPS Frames per second to extract from the video.
Max Size Downscale video frames to fit within a square this many pixels tall and wide.
Other Args Additional space-separated arguments to pass to splat.py extract.
filter-blurry Keep % Percentage of frames to keep after checking for blurriness. If this is 90, the blurriest 10% of frames will be dropped. A value of 100 will skip the filter-blurry step.
Other Args Additional space-separated arguments to pass to splat.py filter-blurry.
feature-extractor Other Args Additional space-separated arguments to pass to colmap feature_extractor.
feature-matcher Subcommand Which colmap subcommand to use for feature matching.
Other Args Additional space-separated arguments to pass to colmap.
colmap Other Args Additional space-separated arguments to pass to colmap.
model-converter-1 Other Args Additional space-separated arguments to pass to the first invocation of colmap model_converter.
check-1 Other Args Additional space-separated arguments to pass to the first invocation of check_reconstruction.py.
prune-spikes Other Args Additional space-separated arguments to pass to prune_spikes.py if it is run.
check-2 Other Args Additional space-separated arguments to pass to the second invocation of check_reconstruction.py, after prune-spikes.
glomap Other Args Additional space-separated arguments to pass to glomap if check-2 fails.
model-converter-2 Other Args Additional space-separated arguments to pass to the second invocation of colmap model_converter, after glomap.
check-3 Other Args Additional space-separated arguments to pass to the third invocation of check_reconstruction.py, after glomap or colmap.
prune-tail Tail Pruning Enable or disable tail pruning. Default is enabled. Tail pruning is not run if check-3 succeeds, regardless of this setting.
Other Args Additional space-separated arguments to pass to prune_tail.py.
prepare Other Args Additional space-separated arguments to pass to splat.py prepare.
gsplat Densification Strategy Densification Strategy to use by gsplat simple_trainer script. Default is the same as the original Gaussian Splatting paper, MCMC is from the Markov Chain Monte Carlo paper.
Data Factor Downscale frames by this factor during model training. You will probably want to adjust Max Size rather than this.
Max Steps Number of training steps to run.
SSIM Lambda Controls how much the training loss emphasizes structural similarity (SSIM) versus raw pixel accuracy. Higher values tend to produce smoother, more visually consistent results, while lower values prioritize sharper pixel-level detail. Higher values also tend to produce larger model files.
Other Args Additional space-separated arguments to pass to simple_trainer.py.
splat-transform Other Args Additional space-separated arguments to pass to splat-transform.
Abort After Abort the processing run after this step. Step names are the same as the names in the left column of this table.
Restart At Restart the processing run at this step, using the files in the archived job directory. Step names are the same as the names in the left column of this table.
