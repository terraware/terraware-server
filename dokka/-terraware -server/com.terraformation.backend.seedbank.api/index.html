enum AccessionStateV2 : Enum<AccessionStateV2>

Supported accession states. This is a subset of the values in AccessionState, minus obsolete states that can still appear in accessions' state histories (and thus need to be kept around as AccessionState enum values) but that can no longer be used as the current states for any accessions.
