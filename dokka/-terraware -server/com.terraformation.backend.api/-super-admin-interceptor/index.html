SuperAdminInterceptor

@Named
class SuperAdminInterceptor : HandlerInterceptor(source)

Allows endpoints to be accessible only by super-admin users. This is used to restrict access to the placeholder admin UI.

Any controller class or handler method that has the RequireSuperAdmin annotation will return HTTP 403 Forbidden to the client if the current user isn't a super admin.
