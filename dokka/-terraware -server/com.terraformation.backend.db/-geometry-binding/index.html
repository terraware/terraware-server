class GeometryBinding : Binding<JooqGeometry, Geometry> (source)

jOOQ binding for the JTS Java library's Geometry type hierarchy. Allows application code to read and write GEOMETRY columns.

Geometry values are always transformed to longitude/latitude (SRID 4326) when they are written to the database. The transformation happens on the database server.

Note that Geometry is an abstract class; queries will always return instances of a concrete class such as Point. It is possible for the same GEOMETRY column on a single table to hold geometries of multiple types, that is, a given query might return a mix of Point and Polygon and other geometry objects.
