collation

val Locale.collation: Collation(source)

Returns the PostgreSQL collation identifier to use when sorting text values in a particular locale.

The PostgreSQL docs have more detail about the collation names, but briefly, both RDS and the default Homebrew PostgreSQL builds have the ICU collations built in, so we'll typically use those for languages other than English.

Note that we can't just blindly tack "-x-icu" onto the locale name; it works for some language/country pairs, but if you use a collation name that PostgreSQL doesn't recognize, it's treated as an error.

To get a list of available collations (note that you should do this on an RDS database of the same version we're using in production, as the list of collations can change between PostgreSQL releases):

SELECT collname FROM pg_collation;
