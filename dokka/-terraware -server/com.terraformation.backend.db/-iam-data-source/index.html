IamDataSource

class IamDataSource : HikariDataSource(source)

Data source that authenticates to the database using a temporary token rather than a password. The token is generated dynamically by AWS.

This is only used if the aws profile is active (that is, if the server is launched with SPRING_PROFILES_ACTIVE=aws.) Otherwise, the server authenticates with a password as usual.
