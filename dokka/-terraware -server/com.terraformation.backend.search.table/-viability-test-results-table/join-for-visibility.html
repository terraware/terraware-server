joinForVisibility

open override fun <T : Record> joinForVisibility(query: SelectJoinStep<T>): SelectJoinStep<T>(source)

Adds a LEFT JOIN clause to a query to connect this table to another table to calculate whether the user is allowed to see a row in this table.

This must join to the same table referenced by inheritsVisibilityFrom.

The default no-op implementation will work for any tables that have the required information already, e.g., if a table has a facility ID column, there's no need to join with another table to get a facility ID. The default implementation is only valid if inheritsVisibilityFrom returns null.
