getSearchSummaryStatistics

fun getSearchSummaryStatistics(originalCriteria: SearchNode): AccessionSummaryStatistics(source)

Returns statistics about accessions that match a set of search criteria.

If there are fuzzy search criteria, this method first checks to see if treating them as exact matches returns any results; if not, it does the fuzzy search as requested. This mirrors the behavior of SearchService.search(), such that the summary data will be consistent with the search results.
