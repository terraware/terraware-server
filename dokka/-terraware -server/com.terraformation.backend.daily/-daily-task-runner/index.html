DailyTaskRunner

@ConditionalOnProperty(value = ["terraware.daily-tasks.enabled"], matchIfMissing = true)
@Named
class DailyTaskRunner(clock: Clock, config: TerrawareServerConfig, publisher: ApplicationEventPublisher, systemUser: SystemUser)(source)

Executes all the tasks that need to run on a daily basis. Handles the clock jumping forward by a large amount suddenly, e.g., because DatabaseBackedClock is being used in a test environment.

When the appointed time arrives, publishes DailyTaskTimeArrivedEvent on the application event bus. Spring will look at all the managed beans to find methods annotated with EventListener that take the event class as an argument, and will call those methods; the listeners may return values that are themselves published as events, allowing tasks to depend on other tasks.
