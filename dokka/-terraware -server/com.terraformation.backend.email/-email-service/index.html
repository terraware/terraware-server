EmailService

@Named
class EmailService(config: TerrawareServerConfig, freeMarkerConfig: Configuration, parentStore: ParentStore, sender: EmailSender, userStore: UserStore)(source)

Renders email messages from templates and sends them to people using the configured mail server.

Email templates are loaded from the src/main/resources/templates/email directory. Please see the README file in that directory for more details.

This class does not interact with any email services; that's done in EmailSender.
