Turtle

class Turtle(start: Point, crs: CoordinateReferenceSystem = CRS.decode("EPSG:${SRID.LONG_LAT}", true))(source)

Constructs polygons using a sequence of movements. The movement distances are always in meters.

The term "turtle" is from turtle graphics.

Initially, the various movement functions only move the turtle but don't add edges to the in-progress polygon. In turtle-graphics terminology, the turtle's pen is up initially. Move the turtle to the initial vertex of the polygon then call startDrawing to put the pen down and start drawing the polygon. The square and rectangle methods automatically start drawing.
