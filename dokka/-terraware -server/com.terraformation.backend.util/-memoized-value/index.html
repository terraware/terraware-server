MemoizedValue

class MemoizedValue<T : Any>(source)

Holds a value that should be computed on first access and then reused after that. This is similar to the val x: Type by lazy {} construct, but allows the computation function to live at the point of access rather than the point of declaration of the variable, which can sometimes improve clarity.
