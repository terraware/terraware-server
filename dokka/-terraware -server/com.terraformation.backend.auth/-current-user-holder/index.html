CurrentUserHolder

object CurrentUserHolder(source)

Makes the current TerrawareUser available to application code. This is modeled after Spring's SecurityContextHolder class: it uses a thread-local variable to hold the model. The variable is populated before the request handler is invoked, then cleared afterwards.
