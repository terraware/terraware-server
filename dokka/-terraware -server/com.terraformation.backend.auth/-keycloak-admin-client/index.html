interface KeycloakAdminClient(source)

A client for the Keycloak admin API. The client handles authenticating to Keycloak; callers only need to concern themselves with the contents of the API requests and responses.

This is modeled after the official Keycloak admin client library, but with a much smaller surface area (it only covers admin functions we actually use).
