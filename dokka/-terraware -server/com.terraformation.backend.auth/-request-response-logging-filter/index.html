RequestResponseLoggingFilter

class RequestResponseLoggingFilter(requestLogConfig: TerrawareServerConfig.RequestLogConfig) : Filter(source)

Logs request and response bodies from users whose lowercase email addresses match a regular expression. This filter should come after TerrawareUserFilter so that CurrentUserHolder is initialized.

Only JSON and HTML form payloads are logged, and only the first 10000 bytes of long payloads are logged. Requests from device manager users are never logged.

Payloads are logged using MDC rather than included in the message text so that they show up as discrete values in structured logs without requiring extra parsing.
