flattenForCsv

fun flattenForCsv(): SearchResults(source)

Turns nested field values into CRLF-delimited strings suitable for exporting to a CSV file.

For example, if the client searched for fields foo.bar and foo.oof and the search results are

[
  {
    "foo": [
      { "bar": "bar 1", "oof": "oof 1" },
      { "bar": "bar 2"  }
    ]
  }
]

this function will return

[
  {
    "foo.bar": "bar 1\r\nbar 2",
    "foo.oof": "oof 1"
  }
]
