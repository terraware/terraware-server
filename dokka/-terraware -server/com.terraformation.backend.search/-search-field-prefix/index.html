SearchFieldPrefix

data class SearchFieldPrefix(val root: SearchTable, val sublists: List<SublistField> = emptyList())(source)

A partial location of a search field in the application's data hierarchy.

The full path of a search field consists of a root, zero or more sublists, and a scalar field. This class is where the root and the sublists live.

See SearchFieldPath for more details about how prefixes are used.
