weightFields

fun weightFields(fieldNamePrefix: String, quantityField: TableField<*, BigDecimal?>, unitsField: TableField<*, SeedQuantityUnits?>, gramsField: Field<BigDecimal?>): Array<SearchField>(source)

Returns an array of SearchFields for a seed quantity: one for each supported weight unit, one for the quantity, and one for the units. This should be expanded into the search table's field list with the * operator so we always have a consistent set of fields for weight searches.

Parameters

fieldNamePrefix

The capitalized units name is appended to this to form the field name. If this is an empty string, the resulting field name will be the non-capitalized units name. For example, if this is "foo", the grams field will be "fooGrams", but if this is "", the grams field will be "grams".
