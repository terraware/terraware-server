AtlassianHttpRequest

interface AtlassianHttpRequest<T>(source)

Every Atlassian API call will implement this interface. The intended workflow is that every time an API call is made, an instance of this interface will be created. The AtlassianHttpClient will invoke the buildRequest method so the caller will not be able to erroneously or maliciously set the request body.
