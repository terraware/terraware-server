DeviceManagerUser

data class DeviceManagerUser(val userId: UserId, val authId: String, parentStore: ParentStore, permissionStore: PermissionStore) : TerrawareUser, UserDetails(source)

Details about the user who is making the current request and the permissions they have. This always represents a device manager client.

Device managers have a limited set of permissions; they can only do things related to managing and monitoring devices.

See also

IndividualUser

for a discussion of how to use this class; for the most part this acts the same, but with a more restricted set of permissions.
