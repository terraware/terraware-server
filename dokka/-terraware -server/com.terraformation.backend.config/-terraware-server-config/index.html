Application-specific configuration options. These are populated by Spring's configuration properties system from a few sources, the most relevant ones being application.yaml files and environment variables. In the YAML files, these are all under a top-level terraware key. In the environment, these are all pulled from environment variables with a TERRAWARE_ prefix in their names.
