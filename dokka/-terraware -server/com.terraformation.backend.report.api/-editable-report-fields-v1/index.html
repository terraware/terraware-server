EditableReportFieldsV1

interface EditableReportFieldsV1 : EditableReportFields(source)

Declares the names and types of the editable fields in a version 1 report. Both the read and write payloads implement this to ensure they stay in sync.

Generally, all the fields here should be nullable since we need to be able to handle saving work in progress before all the required information is filled out.
