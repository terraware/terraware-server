interface PutReportPayload : EditableReportFields(source)

Base interface for PutReportRequestPayload.report. The mapping between version numbers and concrete payload classes is defined here.

Generally speaking, there won't be too many implementations of this interface, because we always migrate in-progress reports to the latest version. But just after we roll out a new report version, there might be users who were in the middle of editing a report. When they hit the save button, we'll get a PUT request that uses the previous version. We need to keep older versions around for long enough to handle those cases, but not after that.
