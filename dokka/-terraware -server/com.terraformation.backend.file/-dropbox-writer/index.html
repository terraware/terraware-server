DropboxWriter

@Named
class DropboxWriter(config: TerrawareServerConfig)(source)

Writes files to Dropbox. API credentials must be configured in TerrawareServerConfig. The following Dropbox permissions are required:

  • files.metadata.write

  • files.content.write

  • sharing.read

  • sharing.write
