compareTo

open operator override fun compareTo(other: AccessionHistoryModel): Int(source)

Compares two history models for sorting purposes. We sort history in reverse date order, with each day's history sorted in reverse createdTime order. We do it this way rather than just sorting by createdTime because some kinds of history can be backdated, e.g., on Tuesday you can tell the system that a withdrawal happened on Monday, and it should show up with Monday's date in the history records.
